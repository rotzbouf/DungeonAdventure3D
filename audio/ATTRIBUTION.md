# Audio Attribution

Curated single-file-per-key for `AudioManager` (M11). All sources are CC0
(public domain) — no attribution legally required, but credited here anyway,
mirroring `MyAssets/from_opengameart/ATTRIBUTION.md`. Each source pack/page
was downloaded in full, one file picked per `AudioManager` key, the rest
discarded.
